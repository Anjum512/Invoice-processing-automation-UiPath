# Invoice Processing Automation using UiPath

## Project Overview

This project automates the processing of invoice data using UiPath and Microsoft Excel. The automation extracts invoice information, validates mandatory fields, identifies missing data, and generates a processed report, reducing manual effort and improving accuracy.

## Objective

* Automate invoice data processing.
* Reduce manual data-entry tasks.
* Validate invoice information.
* Generate automated processing reports.

## Technologies Used

* UiPath Studio
* Microsoft Excel
* Robotic Process Automation (RPA)
* Excel Activities

## Features

* Read invoice data from Excel files.
* Validate mandatory invoice fields.
* Identify missing or invalid data.
* Generate processed invoice reports.
* Reduce manual effort and improve accuracy.

## Workflow

1. Read invoice data from an Excel file.
2. Iterate through each invoice record.
3. Validate mandatory fields such as Invoice Number, Vendor Name, and Amount.
4. Mark records as Processed or Invalid.
5. Generate an output report containing processing status.

## Sample Input Fields

* Invoice Number
* Vendor Name
* Invoice Date
* Amount

## Expected Output

The automation generates an output Excel file containing invoice details along with their processing status.
